Detailed Description

Helper class to organize mesh and voxels volume access and build point sequences.

Note
this class is not thread-safe but accessing same volume from different instances is ok Generated from class MR.MeshOnVoxelsT<const MR.Mesh, MR.SimpleVolumeMinMax>. This is the const half of the class.
