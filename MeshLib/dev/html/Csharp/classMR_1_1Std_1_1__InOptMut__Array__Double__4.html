Detailed Description

This is used for optional parameters of class Box_Array_Double_4 with default arguments. This is only used mutable parameters. For const ones we have _InOptConst_Array_Double_4. Usage:

  • Pass null to use the default argument.
  • Pass new() to pass no object.
  • Pass an instance of Box_Array_Double_4/ConstBox_Array_Double_4 directly.
  • Pass new(ref ...) to pass a reference to Array_Double_4.
