Detailed Description

This is used as a function parameter when passing Mut_UndirectedEdgeId by value with a default argument, since trying to use ? instead seems to prevent us from taking its address. Usage:

  • Pass an instance of Mut_UndirectedEdgeId/Const_UndirectedEdgeId to copy it into the function.
  • Pass null to use the default argument
