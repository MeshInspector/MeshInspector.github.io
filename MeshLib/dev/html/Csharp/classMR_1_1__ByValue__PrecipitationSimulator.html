Detailed Description

This is used as a function parameter when the underlying function receives PrecipitationSimulator by value. Usage:

  • Pass an instance of PrecipitationSimulator/Const_PrecipitationSimulator to copy it into the function.
  • Pass Move(instance) to move it into the function. This is a more efficient form of copying that might invalidate the input object. Be careful if your input isn't a unique reference to this object.
  • Pass null to use the default argument, assuming the parameter has a default argument (has ? in the type).
