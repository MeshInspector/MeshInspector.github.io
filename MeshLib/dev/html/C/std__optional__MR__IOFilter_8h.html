typedef struct MR_std_optional_MR_IOFilter MR_std_optional_MR_IOFilter

Stores either a single MR::IOFilter or nothing. Supported MR_PassBy modes: MR_PassBy_DefaultConstruct, MR_PassBy_Copy, MR_PassBy_Move, (and MR_PassBy_DefaultArgument and MR_PassBy_NoObject if supported by the callee).
