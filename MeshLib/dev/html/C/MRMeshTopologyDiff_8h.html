typedef struct MR_MeshTopologyDiff MR_MeshTopologyDiff

this object stores a difference between two topologies: both in coordinates and in topology

if the topologies are similar then this object is small, if the topologies are very distinct then this object will be even larger than one topology itself Generated from class MR::MeshTopologyDiff.
