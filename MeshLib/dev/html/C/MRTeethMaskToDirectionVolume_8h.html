◆ MR_TeethMaskToDirectionVolumeConvertor

typedef struct MR_TeethMaskToDirectionVolumeConvertor MR_TeethMaskToDirectionVolumeConvertor

This class is an alternative to directly invoking meshToDirectionVolume for the mesh retrieved from the teeth mask. It is better because when a single mesh is created from mask, some neighboring teeth might fuse together, creating incorrect mask. This class invokes meshing for each teeth separately, thus eliminating this problem. Generated from class MR::TeethMaskToDirectionVolumeConvertor.
