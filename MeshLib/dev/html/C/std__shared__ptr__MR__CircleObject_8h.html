typedef struct MR_std_shared_ptr_MR_CircleObject MR_std_shared_ptr_MR_CircleObject

Wraps a pointer to a single shared reference-counted heap-allocated MR::CircleObject. Supported MR_PassBy modes: MR_PassBy_DefaultConstruct, MR_PassBy_Copy, MR_PassBy_Move (and MR_PassBy_DefaultArgument and MR_PassBy_NoObject if supported by the callee).

Overwrite the existing instance, taking ownership of an existing pointer. The previously owned object, if any, has its reference count decremented. Parameter _this can not be null. ◆ MR_std_shared_ptr_MR_CircleObject_use_count()

MRC_API int MR_std_shared_ptr_MR_CircleObject_use_count ( const MR_std_shared_ptr_MR_CircleObject * _this)

How many shared pointers share the managed object. Zero if no object is being managed. This being zero usually conincides with MR_std_shared_ptr_MR_CircleObject_get() returning null, but is ultimately orthogonal. Note that in multithreaded environments, the only safe way to use this number is comparing it with zero. Positive values might change by the time you get to use them. Parameter _this can not be null. It is a single object.
