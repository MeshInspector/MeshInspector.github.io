◆ isBdEdge()

MRMESH_API bool MR::MeshTopology::isBdEdge ( EdgeId e,
const FaceBitSet * region = nullptr ) const

isBdEdge(e) returns true, if the edge (e) is a boundary edge of the mesh: (e) has a hole from one or both sides. isBdEdge(e, region) returns true, if the edge (e) is a boundary edge of the given region: (e) has a region's face from one side (region.test(f0)==true) and a hole or not-region face from the other side (!f1 || region.test(f1)==false).
