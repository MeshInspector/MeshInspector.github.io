typedef struct MR_std_string_view MR_std_string_view

A non-owning string view. Not necessarily null-terminated. Supported MR_PassBy modes: MR_PassBy_DefaultConstruct, MR_PassBy_Copy, MR_PassBy_Move, (and MR_PassBy_DefaultArgument and MR_PassBy_NoObject if supported by the callee).
