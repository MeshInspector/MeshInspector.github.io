typedef struct MR_MeshDiff MR_MeshDiff

this object stores a difference between two meshes: both in coordinates and in topology

if the meshes are similar then this object is small, if the meshes are very distinct then this object will be even larger than one mesh itself Generated from class MR::MeshDiff.
