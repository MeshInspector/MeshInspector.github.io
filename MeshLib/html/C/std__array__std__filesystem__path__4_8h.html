typedef struct MR_std_array_std_filesystem_path_4 MR_std_array_std_filesystem_path_4

A fixed-size array of std::filesystem::path of size 4. Supported MR_PassBy modes: MR_PassBy_DefaultConstruct, MR_PassBy_Copy, MR_PassBy_Move (and MR_PassBy_DefaultArgument and MR_PassBy_NoObject if supported by the callee).
