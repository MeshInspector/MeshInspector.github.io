typedef struct MR_expected_MR_Vector3i_std_string MR_expected_MR_Vector3i_std_string

Stores either a MR::Vector3i that represents success or a std::string that represents an error. Supported MR_PassBy modes: MR_PassBy_DefaultConstruct, MR_PassBy_Copy, MR_PassBy_Move (and MR_PassBy_DefaultArgument and MR_PassBy_NoObject if supported by the callee).
