typedef struct MR_PointToPointAligningTransform MR_PointToPointAligningTransform

This class and its main method can be used to solve the problem well-known as the absolute orientation problem. It means computing the transformation that aligns two sets of points for which correspondence is known. Generated from class MR::PointToPointAligningTransform.
