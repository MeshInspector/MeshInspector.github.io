MRC_CUDA_API int MR_std_shared_ptr_MR_Cuda_PointsToMeshProjector_use_count ( const MR_std_shared_ptr_MR_Cuda_PointsToMeshProjector * _this)

How many shared pointers share the managed object. Zero if no object is being managed. This being zero usually conincides with MR_std_shared_ptr_MR_Cuda_PointsToMeshProjector_get() returning null, but is ultimately orthogonal. Note that in multithreaded environments, the only safe way to use this number is comparing it with zero. Positive values might change by the time you get to use them. Parameter _this can not be null. It is a single object.
