typedef struct MR_VertCoordsDiff MR_VertCoordsDiff

this object stores a difference between two vectors with 3D coordinates

if the vectors are similar then this object is small, if the vectors are very distinct then this object will be even larger than one vector itself Generated from class MR::VertCoordsDiff.
