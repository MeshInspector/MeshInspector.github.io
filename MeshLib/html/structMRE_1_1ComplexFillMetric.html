Detailed Description

This struct provides complex metric which fines new triangles for:

  1. Angle with neighbors: ( (1-cos(x))/(1+cos(x)) )^4
  2. Triangle aspect ratio: Rabc / (2 rabc)
  3. Triangle area (normalized by max loop edge length^2)
    getTriangleMetric - grows with neighbors angles and triangle aspect ratio (R/2r)
    getEdgeMetric - grows with angle
    See also
    MR::FillHoleMetric
