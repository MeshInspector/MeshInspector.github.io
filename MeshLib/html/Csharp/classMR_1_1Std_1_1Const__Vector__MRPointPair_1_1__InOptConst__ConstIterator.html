Detailed Description

This is used for optional parameters of class ConstIterator with default arguments. This is only used const parameters. For non-const ones we have _InOptMut_ConstIterator. Usage:

  • Pass null to use the default argument.
  • Pass new() to pass no object.
  • Pass an instance of ConstIterator/Const_ConstIterator to pass it to the function.
