)

computes the intersection of the shapes bounded by input 2d contours

Returns
the boundary of the intersection

input contours must be closed within the area of distance map and be consistently oriented (counterclockwise, that is leaving the bounded shapes from the left). the value of params.withSign must be true (checked with assert() inside the function)

Note
that polyline topology should be consistently oriented
