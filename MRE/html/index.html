Preparing Third Parties
Some third parties are taken from vcpkg, while others (missing in vcpkg) are configured as git submodules.

Vcpkg:

  1. Please install vcpkg, and integrate it into Visual Studio (note that vcpkg requires English laguage pack installed in Visual Studio):
    $ git clone https://github.com/Microsoft/vcpkg.git
    $ cd vcpkg
    $ git checkout ea11604579181b0ff790be223ee3ded3068142f9
    $ .\bootstrap-vcpkg.bat
    $ .\vcpkg integrate install (with admin rights)
    More details here: vcpkg
  2. Copy MeshRUs/thirdparty/vcpkg/triplets/x64-windows-meshrus.cmake to vcpkg/triplets folder of vcpkg installation.
  3. Execute install.bat
    $ cd vcpkg # or add vcpcg to PATH
    $ work/MeshRusExt/MeshRUs/thirdparty/install.bat
  4. Open solution file MeshRusExt/source/MeshRusExt.sln in Visual Studio 2019. Build it and run.

Build with CMake on Linux

This installation was checked on Ubuntu 20.04.4 with cmake 3.19.3 and gcc 9.3.0.
Installation process is automated, but takes ~40 minutes if no required packages are already installed..
Install/Build dependencies. Build project. Run Test Application Run the following in terminal:

$ git clone https://github.com/AllWish/MeshRusExt.git
$ cd MeshRusExt
$ ./install.sh
$ ./build.sh
$ ./installMRE.sh #optional
$ ./build/Release/bin/MREApp
Note
./install.sh required root access sometimes during installation. However, it's not recommended to run the whole script with sudo.
Note
You could specify build type to Debug by export MESHRUS_BUILD_TYPE=Debug , Release is default.
