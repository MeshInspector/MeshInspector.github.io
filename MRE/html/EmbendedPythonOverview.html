Python overview

Python is one of the most popular and widely used programming language in scientific researches, machine learning, data analysis and advanced testing.

MeshRUs library allows you to call python scripts directly from the command line or use particular functions by import mrpy.

Usage

MeshRUs provides several usage options

  1. Directly from system python
    import mrpy
    mesh = mrpy.load_mesh("mesh.stl")
    # ...
    # some mesh modifications can be perform here
    # ...
    mrpy.save_mesh("mesh.ply")
  2. Run scritp with MeshRUs executable file (MeshRUs.exe on windows or ./MRApp on linux)
    $ ./MRApp script_path.py
    Note
    Script should still import mrpy module
  3. Run script from c++ code
    MR::runScript("script.py"); // or MR::runString( stringWithScript )
    MR::runScript
    MRMESH_API void runScript(const std::filesystem::path &path)

Exposing

You can expose your c++ functions and classes to python

#include "MRPython.h"
void pythonExample( float arg )
{
doSomething(arg);
}
ADD_PYTHON_FUNCTION (examplePythonFunc, pythonExample);

Functions MeshRUs/pythonScripts folder contain scripts which could be used as examples for developers or testers. Python boost c++ library allows to list all mrpy.* functions with standart print(dir(mrpy)) call. You can list all available python functuins by running python MeshRUs/python_scripts/functions_list.py

Note
Not all functions are available anytime, for example functions that triggers rendering are not available in direct python call, but available with executable run

Some functions are listed here:

Function name Function name c++ Arguments Description
draw pythonDraw bool force: false - draw only if scene is changed call draw function of the viewer
fit_data pythonFitData fit camera zoom and snap camera angle to closest canonical quaternion
load pythonLoad string filename load model of known format
mesh_to_points pythonMeshToPoints bool saveNormals: calculate from tiangles and save normal for each point converts Mesh to PointCloud
sample_points_by_grid pythonSamplePointsByGrid float voxelSize: each voxel of that size could have only one point simplifies PointCloud by sampling points using voxels
sample_points_uniform pythonSamplePointsUniform float radius: min distance between sampled points simplifies PointCloud by sampling points using distance between them
save pythonSave string filename saves selected model in file of known format/extention
select_by_name pythonSelectName string modelName select only models with that name, unselect others
select_by_type pythonSelectType string typeName: {"Meshes", "Points", "Voxels"} select only models with that type, unselect others
unselect_all pythonUnselect unselect all models
clear_scene pythonClearScene remove all models
